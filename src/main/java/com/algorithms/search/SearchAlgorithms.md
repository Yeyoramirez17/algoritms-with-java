# Algoritmos de Busqueda
Los algoritmos de búsqueda son técnicas utilizadas para encontrar un elemento específico dentro de una estructura de datos, como un arreglo, una lista enlazada, un árbol o un grafo. A continuación, se presentan algunos de los algoritmos de búsqueda más comunes:

## Búsqueda Lineal
La búsqueda lineal, también conocida como búsqueda secuencial, es un algoritmo que recorre cada elemento de la estructura de datos hasta encontrar el elemento deseado o llegar al final. Es simple pero ineficiente para grandes conjuntos de datos.

```math
\text{Búsqueda Lineal:} \quad O(n)
```

## Búsqueda Binaria
La búsqueda binaria es un algoritmo eficiente que se utiliza en estructuras de datos ordenadas. Divide repetidamente el espacio de búsqueda a la mitad hasta encontrar el elemento deseado o determinar que no está presente.

```math
\text{Búsqueda Binaria:} \quad O(\log n)
```

## Jump Search
La búsqueda por salto es una mejora de la búsqueda lineal que salta un número fijo de elementos en lugar de recorrer cada uno. Es eficiente para grandes conjuntos de datos ordenados.

```math
\text{Jump Search:} \quad O(\sqrt{n})
```
