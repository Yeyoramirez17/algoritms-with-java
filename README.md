# Estructura de Datos y Algoritmos
Implementación de las diferentes estructuras de datos y algoritmos en el lenguaje Java, con el objetivo de comprender su funcionamiento y aplicabilidad en la resolución de problemas computacionales.

## [Algoritmos de busqueda](./src/main/java/com/algorithms/search/SearchAlgorithms.md)
Los algoritmos de búsqueda son técnicas utilizadas para encontrar un elemento específico dentro de una estructura de datos. 

## [Algoritmos de ordenamiento](./src/main/java/com/algorithms/sort/SortingAlgorithms.md)
Los algoritmos de ordenamiento son técnicas utilizadas para organizar los elementos de una estructura de datos en un orden específico, como ascendente o descendente.
